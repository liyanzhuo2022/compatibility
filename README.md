# **PC Builder ‚Äì Custom PC Compatibility Recommender**

A lightweight prototype web application built with **Python** and **Django** to recommend compatible PC components for custom computer builds.

Developed during Enigma x Megekko Hackathon in Tilburg University as a team project.

---

## üöÄ **Features**

- **Compatibility Matching:**
    
    Backend logic checks CPU, GPU, motherboard, RAM, PSU, and case compatibility based on technical specifications.
    
- **Component Recommendation Engine:**
    
    Suggests valid combinations of parts for a full PC build.
    
- **Simple Frontend UI:**
    
    Built with **Bootstrap** for fast prototyping and clean product display.
    
- **Team Collaboration:**
    
    Developed in a small team during a hackathon, using **GitHub** for version control, issue tracking, and coordination.
